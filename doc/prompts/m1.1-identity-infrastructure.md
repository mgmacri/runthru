- **Do NOT restructure to `lib/shared/`** in this milestone. Just rename identifiers in place. Directory restructuring is incremental and happens as features are touched later.
- **Icon task**: If no source icon assets exist, mark T1.1.1.2.1.1 as blocked and skip. Don't generate placeholder icons.
